# Contributing

Thanks for cloning. Here's how to engage:

## Star first

If this saves you time on a link building campaign, a star helps other SEOs find it.

## Found a bug or bad output?

Open an issue with:
- The target article URL (or describe it)
- Which pitch type misfired
- What the output actually said

## Want to extend it?

Good extension candidates:
- Additional pitch types beyond the current three
- Multi-language outreach support
- Domain authority scoring integration

Open an issue before writing code — scope alignment saves everyone time.

## Questions

Use GitHub Discussions.
